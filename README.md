# todoList
todoを追加し、記録する

# Dependency
使用言語:JavaScript フレームワーク:Vue.js

# Setup
推奨ブラウザ:Chrome,Firefox 他ブラウザは非対応

# Usage
* todo追加  
Dateに期日、Taskにtodoを記載し、add押下
* todo削除  
削除ボタン押下、一括削除ボタンで一括削除できる
* 絞り込み  
作業中、完了で絞り込み可能
